# CG-Chess-Project
This project is a chess game implemented in THREE.js. It includes a graphical user interface (GUI) for players to interact with the game. The game features standard chess rules and allows players to compete against each other or against an AI opponent.
## Features
- 3D chessboard and pieces rendered using THREE.js.
- Interactive GUI for selecting and moving pieces.
- Implementation of standard chess rules.
## Optional Features
- Difficulty levels for AI opponent.
- Environment effects (e.g., lighting, shadows, objects textures).
- Ability to see famous chess matches and explanation of moves in it.
